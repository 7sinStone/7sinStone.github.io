Description

Elissa is a discussion forum for women to ask questions in order to clarify or open debates safely and anonymously.

The Elissa web site consists of two parts:
- The first part is the forum where users can register, create discussions and respond to them using a feedback system.
- The second part is a space for administering the content of the forum. It allows the administrator to manage registered users, discussions, answers and categories.

Forum features:
The user can benefit from the functionalities of our forum only after having created an account and authenticate with. And to be able to authenticate, the user must first activate his account via a confirmation mail.
Once authenticated, it can then:
Create a discussion.
Reply to one's own discussion or to another created by another user.
Modify or delete his own discussions.
Modify his answers.
Like a discussion or an answer.
Alert the administrator if someone has not complied with the charter by using insults, abuse, sexual content ...
Search the discussions by keywords.
Consult his information and edit his photo.
Receive notifications if someone has responded to a discussion he has created or to a discussion he has answered before.
It is also necessary that any visitor of our forum be able to consult the rules of use and the FAQs.

The administration panel features:
Through a forum administration dashbord, the administrator can:
Manage the different categories of the forum; He can add a new categorie, edit or Delete an existing one. It can also consult the list of categories.
Consult the number of discussions that were created during the current month or the current year by category.
Consult the discussion list by category.
Manage the discussions; He can view the list of all discussions and delete one or more discussions.
Consult statistical data on the total number of discussions, the number of active discussions in the current month, or the number of discussions created per month or year.
Consult the informations of a discussion.
Delete answers if they contain insults, abuse, sexual content ...
Manage the users registered in the forum; He can consult the list of all registred users. It can also delete or ban one or more users.
Consult statistical data on the total number of users using the forum, the number of users registered in the current month, or the number of users registered per month or per year.
Consult the profile of a user and access to his informations and his recent activities.
Receive alerts from users of a usage rule that is not respected by another user.
