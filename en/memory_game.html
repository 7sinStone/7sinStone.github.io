Memory game

Description

It is a simple memory game designed with html5 and javascript library CreateJS.
It is a game created for the association JCI (for "Jeune Chambre Internationale") of Cathage with a goal to make known the global goals for sustainable development.
you can find a demo of the game in : Demo
The source code is distributed using Github

Keywords: traffic generation, Shinken, Iperf3, D-ITG Python, RPC, Pyro4, Flask, web socket
