Mobile application “Didon”

Description

Didon is an android application that allows people to register for the contest launched by didon association.
The application interacts with a REST API created using a simple PHP framework called SLIM.

The application's features
With this application, the user can:
Register for the contest.
See his informations.
Change his informations.
Consult the contest description.
Consult the description of the association "DIDON".
Consult the contest rules.
